# 未来予測

!!! abstract "この章について"
    この章では、未来予測のプリセットの使い方を扱います。

## 未来予測のプリセット

<!-- 画像さしこみ: 未来予測のプリセット -->

### 操作手順

1. ヘッダーの出生データピッカーで、未来予測したい出生データを選択します。
2. メニューから **未来予測** を開きます。
3. **イベントプリセット** から、「**軸通過**」「**成功相**」「**恋愛運**」「**大きな出来事**」のいずれかを選びます。
4. **開始年月日** を入力します。
5. 期間は、開始年月日から **7日 / 1カ月 / 1年 〜 50年** のボタンをクリックするか、**終了年月日** を入力して指定します。
6. **経過地** は、プリセットでは入力不要です（これらのイベントプリセットは経過地に影響されないためです）。
7. 「**計算**」ボタンを押すと、対象の時期が一覧で表示されます。

!!! note "プリセットの説明までを扱います"
    ここでは初級のため、未来予測は **プリセットの説明まで** とします。個別手法の詳細はスタナビ上級編で扱います。

### 補足説明

- **各イベントプリセットの意味**
    - **軸通過**: 重要なポイントに天体が重なる、重要な時期を抽出します。
    - **成功相**: 主に仕事での成功のチャンスが訪れやすい時期を抽出します。
    - **恋愛運**: 恋愛運が訪れやすい時期を抽出します。
    - **大きな出来事**: 天体がメジャーアスペクトを作る時期を抽出します。
- **プリセットの保存**: 期間や天体などの条件を設定し、それをご自分のプリセットとして保存できます。設定のプリセットと同じように「**上書き保存**」または「**別名で保存**」で保存でき、次回から同じ条件をすぐに呼び出せます。
- 未来予測機能は、Plus 以上のプランでご利用いただけます。
