# StarNavigator 使い方マニュアル

このサイトでは、スタナビの **使い方（操作方法）** のみを扱います。占星術の解釈・知識そのものは、セミナー等で扱います。

- [三重円](triple-chart.md) — 経過図（N・P・T）
- [二重円](double-chart.md) — シナストリー・新月図を重ねる
- [未来予測](forecast.md) — 未来予測のプリセット
- [リターン図](return-chart.md) — ソーラーリターン図
