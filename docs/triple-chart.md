# 三重円

!!! abstract "この章について"
    この章では、三重円（N・P・T）の作成・表示・設定をまとめます。

## 三重円の作り方

<!-- 画像さしこみ: 三重円の作り方 -->

### 操作手順

1. ヘッダーで **出生データ** と **プリセット** を選びます。
2. メニューから「**三重円**」を選びます。
3. 経過日時はデフォルトで **現在日時 × デフォルト観測地** がセットされているので、必要なら変更します。
4. 進行法は **セカンダリープログレッション** または **ソーラーアーク** を選択します。
5. 「**チャートを表示**」を押すと、左側に三重円、右側にデータパネルが表示されます。データパネルに表示されるのは、プリセットで選択されている天体やアスペクトです。
6. **度数** にチェックを入れると、チャート上に各天体の度数が表示されます。
7. **複合アスペクト** にチェックを入れると、複合アスペクトが計算されます。計算に多少時間がかかるので、必要なときだけ ON にしてください（複合アスペクト表示は Basic 以上のプラン）。

### 補足説明

- **俯瞰タブ** では、旧スタナビと同じレイアウトで出生・進行・経過の天体情報が一覧で並び、その下にアスペクトグリッドが表示されます。
- 一重円と同じく、チャート上の天体やサイン・ハウスをクリックすると意味が表示されます。
- その他のデータタブ（天体・ハウス・アスペクト・分析・ハーフサム）の使い方は、**4-2. チャート表示とパネルの説明** と同じです。

## 天文暦（エフェメリス）の表示

<!-- 画像さしこみ: 天文暦（エフェメリス）の表示 -->

### 操作手順

1. 三重円画面で「**天文暦**」ボタンを押します。
2. ダイアログが開き、設定した経過日の前後の天文暦（各天体の位置）を日別で確認できます。
3. **逆行中** の天体は緑文字 + R、**日曜** は赤、**土曜** は青で表示されます。

### 補足説明

- 天文暦の度数表記は **60進法固定** です。ヘッダーの 10進法／60進法 切り替えは、天文暦ダイアログには適用されません。

## 表示設定

<!-- 画像さしこみ: 表示設定 -->

### 操作手順

1. 三重円画面で「**表示設定**」ボタンを押します。
2. **N（ネイタル）／P（進行）／T（経過）** ごとに天体とアスペクトを ON／OFF して、この三重円だけの内容を調整できます。
3. 変更内容は「**上書き保存**」で現在のプリセットに反映するか、**プリセット名を入力して「新規作成」** で別プリセットとして保存できます。

### 補足説明

- 一重円の表示設定（**4-4. 表示設定**）と同じ UI で、N／P／T を変更できます。Plus 以上のプランでご利用いただけます。
